# Legal And Data Policy

Reports are market commentary and source-backed analysis, not financial advice.

This repository stores citation metadata and links. It must not store copied full articles, paywalled article text, credentials, private user content, or local runtime state.

If a future report uses non-public data providers, redistribution and citation terms must be reviewed before any provider-derived data is committed.

Public GitHub contributions are not accepted during this phase. Questions and corrections belong on the linked publication channel and are recorded here only as sanitized metadata when needed.
