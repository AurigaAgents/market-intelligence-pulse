# Contributing

Market Intelligence Pulse is currently an agent-operated publication archive. Public GitHub contributions are not accepted during this phase.

Please do not open issues or pull requests for market corrections, questions, or comments. Use the linked publication channel instead. If a substantive correction is accepted, Market Intelligence Pulse records a sanitized Q&A or erratum reference in `qa.json`.

Automation ignores issues and pull requests that do not carry the managed project-control labels and responsibility block.
